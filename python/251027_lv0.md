### (6065). 가장 작은 값 또는 큰 값 출력하기
```python
# 최대값 코드
a, b, c = map(int, input().split())

if a >= b and a >= c:
    print(a)
elif b >= a and b >= c:
    print(b)
else:
    print(c)
# 와 동일한 코드
print(max(a, b, c))

# 최소는
print(min(a, b, c))

```
### (6066). 정수 세 개 입력 받아 짝 홀 출력(for문 이용해보기)
```python
for i in map(int, input().split()):
    if i % 2 == 0 :
        print("even")
    else :
        print("odd")
```
