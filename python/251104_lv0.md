### (6072). 카운트 다운 ⭐
```python
n = int(input())
# n -= 1      (n보다 1 작은 거에서부터 세고 싶을 때 )
while n > 0 :      # 0까지 세고 싶을때는 n >= 0 으로 하
    print(n)
    n -= 1        # 앞에 먼저 - 쓰는거 잊지 말기 ✔️

# 두번 째는 for 문 이용해보기
n = int(input())
for i in range(n, 1, -1)     # n부터 1까지 -1씩 되면서 출력하기 ✔️
    print(i)
```
#### for 문 예시 문제 
```python
# 증가(감소) 간격 지정
for i in range(1, 6, 1):   # 1에서 5까지 1씩 증가해서 나옴
    print(i)

# 문자열에서 사용 
for ch in "seoan"
    print(ch)   # 한글자씩 출력됨

# 합 구하기 🤔
total = 0     # 빈 상자를 구한 것임 
for i in range(1,6)
    total += i
print(total)
```
### (6074). 알파벳 입력받아 그 알파벳까지 출력
```python
# for 문으로 풍기
ch = input()

for i in range(ord('a'), ord(ch) + 1):    # ord는 a의 아스키 코드 값
    print(chr(i), end=' ')                # ⭐ chr(i)는 숫자를 다시 문자로 변환, end는 줄바꿈 대신 띄어쓰기로 출력

# while문으로도 해보기 ✔️
ch = input()

i = ord('a')
while i <= ord(ch):
    print(chr(i), end='')
    i += 1
```

### (6077). 정수 입력받아 그 수까지 짝수만 더하기
```python
# for 문 버전
n = int(input())
total = 0

for i in range(2, n + 1, 2):
    total += i

print(total)

# while 문 버전
n = int(input())
total = 0
i = 2   # 홀수 일 시 i=1

while i <= n :
    total += i
    i += 2
print(total)
```

### (6079). 언제까지 더해 😒⭐
**1, 2, 3, …를 차례로 더하다가 합이 입력값 n과 같아지거나 넘어가는 순간의 “마지막에 더한 수”**를 출력
- “작을 때까지 더한다” → while total < n:
- “커지거나 같아지면 멈춘다” 가 핵심
```python
n = int(input())

total = 0
i = 0
while total < n:   # 합이 n보다 작은 동안 계속 더한다 ✔️
    i += 1         # 다음 자연수
    total += i     # 합에 더하기

print(i)           # 합이 n에 도달(또는 초과)했을 때 마지막으로 더한 수
```
